# 量化策略开发项目

## 项目说明
这是一个Python量化策略开发项目，包含多种交易策略的实现和回测。

## 项目结构
- `strategies/` - 策略实现
- `data/` - 数据模块
- `backtest/` - 回测引擎
- `notebooks/` - 分析笔记本
- `config/` - 配置文件

## 使用方法
1. 克隆项目：`git clone https://github.com/你的用户名/quant_strategies.git`
2. 安装依赖：`pip install -r requirements.txt`
3. 运行策略：`python strategies/ma_strategy.py`

## 开发规范
- 新功能在feature/*分支开发
- 提交信息按规范格式
- 主分支main保持稳定
